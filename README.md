Tic Tac Toe is a simple two-player game played on a 3×3 grid.
Players take turns marking a cell with X or O.
The first player to align three marks horizontally, vertically, or diagonally wins the game.
# Technologies Used
Python 3
Console/Terminal
# Game Rules
The game is played on a 3×3 grid.
Player 1 uses X
Player 2 uses O
Players enter position numbers (1–9) to place their mark.
The first player to get 3 marks in a row wins.
If all 9 cells are filled and no player wins, the game ends in a draw.
# Features
Two-player mode
Input validation
Win detection
# Learning Outcome
Learned how to implement game logic and winning conditions by developing a simple Tic Tac Toe game.
